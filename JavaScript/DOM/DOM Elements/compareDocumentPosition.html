HTML DOM Element compareDocumentPosition()

Where is "p1" compared to "p2" :

This is a paragraph.

This is another paragraph.

The position of the two paragraphs is:

Possible value:
1:The two nodes do not belong to the same document.
2:p1 is positioned after p1.
4:p1 is positioned before p2.
8:p1 is positioned inside p2.
16:p2 is positioned inside p1.
32: the two nodes are attributes on the same element.

The return value can also be a combination of values.
Return value 20 means that p2 is inside p1(16) ANd p1 is positioned before p2 (4).
